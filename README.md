# Requirements
* Docker
* .NET 8.0
# How to compile
Run `$ docker compose up -d`

Run `$ dotnet run --project src/MatchService.csproj`

Go to `http://localhost:5242` to see the api in swagger

Also you can go to `http://localhost:5050` to see PGAdmin
